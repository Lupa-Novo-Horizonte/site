Acesso a API

Para ter acesso as APIs do projeto e utilizar os dados em tempo real direto do banco de dados, envie um e-mail requisitando um login. Através deste login será possível autenticar e gerar o token para as requisições.

Abaixo listamos o passo a passo:

  1. Acesse a aba SOBRE e logo após a aba CONTATO. Envie uma mensagem requisitando o acesso e retornaremos a mensagem com um usuário e senha.
  2. Execute a API de login utilizando os dados de acesso compartilhado.
  3. Utilize o token gerado na API de login nas demais APIs

Na API login utilize o usuário e senha para gerar o token. Guarde o número gerado no campo TOKEN para demais requisições.
